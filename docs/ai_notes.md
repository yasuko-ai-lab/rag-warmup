# AI Basics

## What is AI?
Artificial Intelligence (AI) refers to systems that can perform tasks that normally require human intelligence.

## What is Machine Learning?
Machine Learning is a subset of AI where systems learn patterns from data instead of being explicitly programmed.

## What is a Large Language Model?
A Large Language Model (LLM) is trained on large amounts of text to predict the next word in a sequence.

## What is RAG?
Retrieval-Augmented Generation (RAG) combines document retrieval with text generation to answer questions based on external knowledge.
